## [2026-05-06] fix | ai-silent-failure

Kaynak: frontend/src/stores/chatStore.ts, frontend/src/api/aiClient.ts, frontend/src/components/ai/ChatWidget.tsx
Sorun: AI asistan kimi zaman cevap vermiyor, hata da vermiyor, ekran boş kalıyor.
Kök neden:
1. `startNewSession` hata fırlatmıyor, `sendMessage` sessionId olmadan çağrılıyordu.
2. `streamChat` içinde timeout mekanizması yoktu, backend yanıt vermezse stream sonsuza kadar bekliyordu.
Çözüm:
- `startNewSession` artık session ID döndürüyor ve hata fırlatıyor.
- `sendMessage` session ID kontrolü yapıyor, yoksa hata mesajı gösteriyor.
- `streamChat` 90 saniyelik timeout ekledi (AbortController ile).
- `handleSend` session oluşturma hatalarını yakalıyor.
Dokunulan dosyalar:
- chatStore.ts: startNewSession return ekledi, sendMessage session kontrolü
- aiClient.ts: 90sn timeout, cleanup fonksiyonu
- ChatWidget.tsx: handleSend try-catch bloğu
