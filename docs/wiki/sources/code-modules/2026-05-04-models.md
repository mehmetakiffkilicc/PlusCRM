# models — Django Veri Modelleri

**Özet:** Projenin tüm kalıcı varlıklarını tanımlayan Django ORM model dosyasıdır. 10 model içerir; DataSource/Dashboard/Widget kullanıcı verisi + analitik altyapısını, AISession/AIMessage/AIAuditLog/AINotification/AIDashboard/ScheduledCampaign yapay zeka alt-sistemini, SystemSetting ise sistem konfigürasyonunu temsil eder.
**Kütüphaneler:** Django ORM, PostgreSQL JSONField
**Bağlantılar:** [[entities/datasource-modeli]], [[entities/dashboard-modeli]], [[entities/widget-modeli]], [[entities/aisession-modeli]], [[entities/ainotification-modeli]], [[entities/scheduledcampaign-modeli]], [[entities/aidashboard-modeli]], [[entities/serializers]]

## Modeller

| Model | Anahtar Alanlar | Amaç |
|---|---|---|
| DataSource | user FK, type (csv/json/database), data JSONField, column_mapping | Yüklenen veri kaynağı |
| Dashboard | user FK, name, description | Kullanıcı panel konteyneri |
| Widget | dashboard FK, data_source FK, type (line/bar/pie/scatter/heatmap), aggregation | Panel içi grafik |
| SystemSetting | key unique, value JSONField | Anahtar/değer sistem ayarı |
| AISession | user FK, model_name (default: gemini-2.0-flash), total_tokens, total_cost | AI sohbet oturumu |
| AIMessage | session FK, role, content, tool_calls JSONField, input/output_tokens | AI mesajı |
| AIAuditLog | user FK, session FK, tool_called, prompt_hash, response_hash | AI araç çağrı audit |
| AINotification | user FK, title, message, type (info/warning/success/critical), is_read | Kullanıcı bildirimi |
| ScheduledCampaign | user FK, segment, channel (sms/email/push), scheduled_at, status | Zamanlanmış kampanya |
| AIDashboard | user FK, name, config JSONField, is_favorite | AI üretimli panel |
