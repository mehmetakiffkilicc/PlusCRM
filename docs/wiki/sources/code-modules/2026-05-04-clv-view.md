# clv-view — Customer Lifetime Value Analizi

**Özet:** Müşteri yaşam boyu değerini (CLV) hesaplar; iki mod destekler: CSV/JSON datasource ve database datasource. 5 dakikalık dahili cache ile drill-down sorguları arasında tutarlılık sağlar.
**Kütüphaneler:** DRF, Python in-process cache
**Bağlantılar:** [[entities/analytics-clv-view]], [[concepts/clv]], [[entities/datasource-modeli]]

## Dahili Cache Mekanizması
```python
_clv_internal_cache = {}        # Dict: cache_key → {segments, timestamp}
_clv_internal_timeout = 300     # 5 dakika TTL
```
Özet ve detay view'ları aynı cache'i paylaşır — drill-down sırasında segmentler değişmez (tutarlılık).

## İki Mod
- **CSV/JSON modu**: `DataSource.data` JSONField'dan yükler, Python'da hesaplar.
- **Database modu**: `data_type == 'database'` veya isimde `sal/sat` geçiyorsa aktif → doğrudan SQL sorgusu.

## `parse_amount()` Yardımcısı
Türkçe ondalık (`1.234,56` → `1234.56`) ve çeşitli format farklılıklarını normalize eder.
