# app-tsx — React Router ve Uygulama İskeleti

**Özet:** Uygulamanın tüm sayfalarını `React.lazy` ile kod-bölümleyerek yükleyen ve route yapısını tanımlayan kök bileşendir. `useAuthStore`'dan auth durumu okur; authenticated kullanıcılar `DashboardLayout` içinde, diğerleri `/giris`'e yönlendirilir.
**Kütüphaneler:** React 18, React Router v6, Zustand
**Bağlantılar:** [[entities/app-router]], [[entities/store-auth]], [[entities/dashboard-layout]]
