# urls — API Endpoint Haritası

**Özet:** Projenin tüm REST endpoint'lerini tek dosyada tanımlayan Django URL konfigürasyonudur. 100+ endpoint 8 gruba ayrılmıştır: Auth, DataSources, CRM Analytics, Kampanya Önerileri, Dashboard/Widget, Panel-SQLite Direct, Sistem, AI Asistan.
**Kütüphaneler:** Django URL dispatcher, DRF
**Bağlantılar:** [[entities/django-core]], [[entities/auth-views]], [[entities/analytics-rfm-view]], [[entities/llm-context-builder]], [[syntheses/analytics-istek-akisi]]

## Endpoint Grupları

### Auth (`/api/auth/`)
| Path | View | Amaç |
|---|---|---|
| `auth/kayit/` | RegisterView | Kullanıcı kaydı |
| `auth/giris/` | LoginView | JWT token alma |
| `auth/profil/` | ProfileView | Profil oku/güncelle |

### CRM Analytics (`/api/veri-kaynaklari/<id>/`)
RFM, Churn, CLV, Segmentasyon, Markalar, Ürün Portal, Kategori Raporu, Müşteri Listesi/Detay, Kohort, Ürün Birliktelik, Marka Sadakati, Enflasyon, Rakip Riski, Hane Analizi — hepsi `data_source_id` parametresiyle ayrıştırılır.

### Panel SQLite Direct (`/api/panel/`)
KPI, trend, karşılaştırma, segment, filtre endpoint'leri — Postgres yerine SQLite cache'ten okur. Hızlı pano yüklemesi için.

### AI Asistan (`/api/ai/`)
Sohbet (stream), yeni oturum, geçmiş, silme, özet, müşteri profili, NBA, varyant üretimi, anomali tespiti, haftalık brifing, bildirimler, kampanya planlama, AI paneller.

### Sistem
Sistem logları, cache yenileme (`/api/sistem/cache-yenile/`), ayarlar.

## Decisions
- [[decisions/karar-sqlite-direct-kpi]]: `/api/panel/` endpoint'leri SQLite direct — Postgres yük azaltma.
