# rfm-view — RFM Segmentasyon Analizi

**Özet:** Recency/Frequency/Monetary skorlarına dayalı 11 segmentli müşteri sınıflandırma endpoint'i. SQLite/Postgres db_engine abstraksiyonu üzerinden çalışır; analitik sonuçlar `base.py` cache altyapısıyla saklanır.
**Kütüphaneler:** DRF, SQLite3, `analytics.base` cache yardımcıları
**Bağlantılar:** [[entities/analytics-rfm-view]], [[concepts/rfm-analizi]], [[entities/datasource-modeli]]

| Segment | Renk | Aksiyon |

| 01-) Şampiyonlar | #10b981 | VIP programı, özel ayrıcalıklar |

## Mimari Notlar
- `db_engine.ph()` ile parametre placeholder'ı (`?` veya `%s`) abstract edilmiş.
- `get_cached_data` / `set_cached_data` (base.py): TTL tabanlı Redis-like in-process cache.
- `detect_columns()`: farklı CSV sütun isimlendirmesini normalize eder.

## Decisions
- Segment isimleri Türkçe ve veritabanındaki isimlendirmeyle (`01-) Şampiyonlar`) birebir eşleşiyor.
